Solapi 메시지 가격 및 운영 구분

기준일: 2026-07-14 · 가격은 Solapi 공개 표준 단가 기준, VAT 별도

우리 원칙: 출결·수업·상담·청구·운영 공지는 알림톡 우선, 승인 템플릿이 없거나 긴급 공지면 SMS/LMS를 사용한다.

BMS_FREE는 사용 금지: 템플릿 없이 자유 발송 가능하지만 100원/건으로 LMS보다 비싸고, 이번 실패의 원인이 된 경로다.

광고성 판단 기준: 수신 방법이 카카오톡인지 문자인지가 아니라 내용이 영리목적 광고성인지 여부다. 정보성 LMS에는 (광고)와 080을 넣지 않는다.

현재 UI 버튼 → 실제 발송 경로

발송·재시도·실패 처리에 직접 관련된 버튼 기준이다. 검색, 닫기, 대상 선택 같은 화면 조작 버튼은 제외했다.

운영 분류

정보성으로 보는 예

  • 출결, 지각, 미등원, 조퇴, 보강 안내
  • 수업 시간 변경, 휴강, 준비물, 과제 안내
  • 상담 일정, 시험 일정, 진단평가 예약 확정
  • 수강료 청구, 미납, 수납 확인
  • 재원생 대상 학원 운영·행사 일정 공지

광고성으로 볼 수 있는 예

  • 방학특강 모집, 신규 강좌 신청 유도
  • 할인, 이벤트, 혜택, 쿠폰, 친구 추천
  • 기존 수강과 직접 관련 없는 상품·서비스 홍보
  • 정보성 공지 뒤에 신청 유도 문구를 붙인 혼합 메시지

우리 시스템 기본값

  • 정보성 알림톡: 승인 템플릿이 있으면 우선 사용
  • 정보성 SMS/LMS: 알림톡 템플릿이 없거나 긴급 공지일 때 사용
  • 홍보성 SMS/LMS: 수신동의 확인 후 (광고)와 080 포함
  • BMS_FREE: 비용과 운영 리스크 때문에 차단
