---
name: 학생 표시는 항상 studentShortLabel
description: 학생 정보를 UI에 노출할 때 학교/학년을 풀어 쓰지 말고 "양정중2" 같은 축약 라벨을 사용한다
type: feedback
---

학생을 화면에 표시할 때(학생 카드 메타, 그룹 헤더, 목록 행, 상세 패널 등) 학교·학년을 풀어서 `${school} · ${grade}` 형태로 쓰지 말고 **반드시 `studentShortLabel(student)`을 사용**한다.

- 함수 위치: `src/shared/firestore-helpers.js`의 `studentShortLabel`
- 출력 예: `"양정중2"`, `"진명여고1"`, `"신목중3"` (학교 + 학부(중/고) + 학년 축약, "여자" → "여" 변환, 중복 '중' 방지)
- 이미 import되어 있는 파일이 많음. 새 사용처에서 import 누락 주의.

**Why:** UI 일관성. 학원 도메인에서 학생을 한눈에 식별하는 표준 표기가 이 축약 형태. 풀어 쓰면 줄바꿈/공간 낭비와 함께 동일 학생이 위치마다 다른 형태로 보이는 인지 부담이 생긴다. 사용자가 명시적으로 "이건 모든 문서에 공통 적용" 이라고 못 박은 규칙.

**How to apply:**
- 새 화면/컴포넌트/리스트 추가 시 학생 표시 부분에 `studentShortLabel`을 기본 선택지로.
- 학생 row 데이터 구조를 만들 때 `shortLabel` 같은 미리 계산된 필드로 넣어두면 호출처가 깔끔 (예: `class-setup.js`의 `makeBaseRow`).
- 그루핑 헤더에 이미 학교/학년이 펼쳐져 표시되는 경우(예: 내신반 계획의 5차원 헤더)는 헤더에서 풀어 쓰는 것이 의미가 있으므로 예외. 단, 학생 한 명을 식별하는 메타에는 축약형 우선.
