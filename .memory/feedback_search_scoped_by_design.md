# 검색은 활성 필터 안에서 동작하는 게 의도된 사양

2026-06-11 UI/UX 감사에서 "결석대장 필터 중 검색하면 결석대장 안에서만 검색됨"을 버그로 보고했으나, **사용자가 의도된 동작이라고 확정**했다.

**Why:** 운영 동선상 현재 보고 있는 목록 안에서 좁혀 찾는 게 기본 사용 패턴.

**How to apply:**
- 검색 스코프를 전역으로 바꾸자고 다시 제안하지 말 것.
- 대신 필터+검색 동시 활성 시 리스트 상단에 범위 안내 배너(`#search-scope-banner`, filter-nav.js renderFilterChips에서 토글) + "전체에서 검색" 버튼(`clearAllFilters()` 재사용)이 떠 있다.
- 단, 반(class) 필터는 검색 중 적용되지 않으므로(list-view.js) 배너 라벨에서 제외돼 있음 — 이 비대칭도 의도.
