## 데이터 소스: daily_checks 컬렉션

### 문서 ID 형식
`{date}_{studentId}_{enrollIdx}` (예: `2026-02-23_student123_0`)

### 주요 필드
- **출석**: `attendance` (출석/결석/지각/조퇴), `attendance_time`, `attendance_reason`
- **숙제 (8영역, 값: O/△/X)**:
  - `hw_reading` (독해), `hw_grammar` (문법), `hw_practice` (실전)
  - `hw_listening` (청해), `hw_extra` (추가), `hw_vocab` (어휘)
  - `hw_idiom` (숙어), `hw_verb3` (3단)
- **테스트 (4영역)**:
  - `test_reading` (독해), `test_grammar` (문법)
  - `test_practice` (실전), `test_listening` (청해)

### 쿼리
```javascript
query(collection(db, 'daily_checks'),
  where('student_id', '==', docId),
  where('date', '>=', startDate),
  where('date', '<=', endDate),
  orderBy('date'))
```

### 집계 로직
1. **출석**: 각 문서의 date + attendance + attendance_reason으로 날짜별 테이블 생성
2. **숙제**: 영역별 O/△/X 각각의 횟수를 집계
3. **테스트**: 영역별 O/△/X 각각의 횟수를 집계 (숙제와 동일 방식)
