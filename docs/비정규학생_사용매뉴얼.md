# 비정규 학생 관리 사용 매뉴얼

> impact7 DSC 출결 관리 시스템 — 비정규 학생 처리 가이드

---

## 1. 비정규란?

**비정규 학생**이란 정규 수업 시간 외에 별도로 학원에 방문하는 학생을 말합니다.

| 유형 | 설명 | 예시 |
|------|------|------|
| 진단평가 | 학원에 등록하지 않은 학생의 시험 | 신규 상담 후 진단평가 예정 |
| 숙제미통과 | 숙제 검사에 불합격하여 재등원 | 영어 숙제 미통과 → 다음 주 등원 |
| 테스트미통과 | 테스트에 불합격하여 재등원 | 단어 시험 미통과 → 재시험 등원 |
| 클리닉 | 보충수업, 재시험 등 추가 등원 | 선생님이 직접 클리닉 일정 추가 |
| 결석보충 | 결석한 수업의 보충 등원 | 지난주 결석 → 보충 수업 |

---

## 2. 비정규 화면 찾기

### 화면 이동 경로
```
출결 → 비정규 탭 클릭
```

### 하위 필터 4가지
비정규 탭 아래에 4개의 필터 버튼이 있습니다:

| 필터 | 표시 내용 | 배지 |
|------|----------|------|
| **진단평가** | 비등록 학생 진단평가 | 미완료/전체 (예: 2/5) |
| **클리닉** | 클리닉 등원 일정 | 미완료/전체 |
| **미통과** | 숙제·테스트 미통과 등원 | 미완료/전체 |
| **결석보충** | 결석 보충 등원 | 미완료/전체 |

---

## 3. 목록 화면 구성

비정규 학생 목록은 4개 구역으로 나뉩니다 (위에서 아래 순서):

### (1) 지연 구역 (빨간색)
- **의미**: 예정일이 지났는데 아직 처리되지 않은 항목
- **표시**: "지연 — 미완료 (N건)"
- **언제 보이나**: 오늘 날짜를 볼 때만 표시됨
- **빨간 뱃지**: "지연 3/11" 처럼 원래 예정일 표시

### (2) 등원전 구역
- **의미**: 아직 학원에 도착하지 않은 학생
- **출석 상태가 "미확인"**인 학생

### (3) 등원후 구역
- **의미**: 학원에 도착한 학생 (출석, 지각, 조퇴)
- **소스별로 그룹**: 클리닉 → 진단평가 → 숙제미통과 → 테스트미통과 → 결석보충

### (4) 확인 완료 구역
- **의미**: 처리가 끝난 항목
- **표시**: "확인 완료 (N건)"
- **상태 표시**: 완료/시행/미완료/미시행/기타 중 어떤 상태로 확인됐는지 표시

---

## 4. 각 항목의 버튼 설명

목록의 각 학생 항목에는 다음 버튼들이 있습니다:

### 상태 버튼 (좌측)
상태를 토글(순환)하는 버튼입니다. 클릭할 때마다 상태가 바뀝니다.

**일반 항목 (숙제미통과, 테스트미통과, 클리닉, 결석보충):**
```
미완료 → 완료 → 기타 → 미완료 → ...
```

**진단평가 항목:**
```
미시행 → 시행 → 기타 → 미시행 → ...
```

> **중요**: 학생이 등원하지 않은 상태에서는 **완료/시행**으로 바꿀 수 없습니다.
> 미등원 시: `미완료 → 기타 → 미완료` (또는 `미시행 → 기타 → 미시행`)

### 확인 버튼 (우측)
현재 선택된 상태를 **최종 확정**합니다. 확인을 눌러야 저장됩니다.

### 초기화 버튼 (확인 완료 후)
이미 확인된 항목을 다시 미완료 상태로 되돌립니다.

### 재지정 버튼 (파란색)
날짜/시간을 변경하여 다른 날로 옮깁니다.

---

## 5. 학생 처리 흐름

### 5.1 학생이 등원한 경우 (출석/지각/조퇴)

```
① 출석 상태 확인: 등원 버튼으로 "출석" 또는 "지각" 체크
② 상태 토글: "완료" (또는 "시행") 선택
③ 확인 클릭: 완료 처리 → 확인 완료 구역으로 이동
```

**확인 완료 후 화면:**
```
[완료] [초기화]    ← 완료 상태 표시 + 되돌리기 버튼
```

### 5.2 학생이 오지 않은 경우 (미등원)

```
① 상태 토글: "미완료" (또는 "미시행") 선택
   ※ 미등원이면 "완료"/"시행"은 자동으로 건너뜁니다
② 확인 클릭: 재지정 화면으로 이동
③ 재지정: 새 날짜와 시간을 입력하여 다음 등원일 지정
```

### 5.3 더 이상 관리하지 않을 경우

```
① 상태 토글: "기타" 선택
② 확인 클릭: 기타 처리 → 확인 완료 구역으로 이동
```

## 6. 유형별 상세 사용법

### 6.1 진단평가

#### 새 진단평가 만들기
1. 비정규 → 진단평가 필터 선택
2. 우측 상단 **[+]** 버튼 클릭
3. 정보 입력:
   - **이름** (필수)
   - **학부모 연락처** (권장) — 입력 시 기존 연락처에서 학교/학년 자동 채움
   - **예정 날짜** (필수)
   - **예정 시간**
   - 소속, 학교, 학부, 학년, 학생연락처, 메모 (선택)
4. **저장** 클릭

#### 진단평가 당일 처리
```
학생 등원 시:
  등원 버튼 → "등원" 체크 → 시행 → 확인 → 완료!

학생 미등원 시:
  미시행 → 확인 → [재지정] 또는 [시험취소] 선택
    → 재지정: 새 날짜/시간 입력 → 저장
    → 시험취소: 기타 처리됨
```

#### 진단평가 정보 수정
1. 목록에서 해당 학생 클릭 → 상세 패널 열림
2. **수정** 버튼 클릭 → 모달 열림
3. 정보 수정 후 저장
4. 수정 이력이 자동으로 기록됨

---

### 6.2 숙제미통과 / 테스트미통과

#### 생성 (자동)
- 정규 학생의 숙제/테스트 검사에서 **미통과** 처리 시
- **등원** 유형을 선택하면 자동으로 비정규 목록에 추가됨
- 예정일과 시간이 자동 설정됨

#### 당일 처리
```
학생 등원 시:
  완료 → 확인 → 완료!

학생 미등원 시:
  미완료 → 확인 → 재지정 모달
    → 새 날짜 입력 (필수)
    → 새 시간 입력
    → 재지정 사유 입력 (선택)
    → 저장
```

#### 재지정 이력
- 재지정할 때마다 이력이 기록됩니다
- 이전 날짜, 새 날짜, 변경자, 사유가 모두 저장됨

---

### 6.3 클리닉

#### 클리닉 추가
1. 정규 학생 상세 패널에서 **클리닉** 카드의 **[+]** 버튼 클릭
2. 날짜, 시간, 사유 입력
3. 자동 저장됨 → 비정규 목록에 표시

#### 클리닉 삭제
- 클리닉 카드의 **[×]** 버튼 클릭
- 과거 기록은 삭제할 수 없음

---

### 6.4 결석보충

#### 생성 (자동)
- 학생이 결석 처리된 후
- 결석대장에서 **보충** 방법을 선택하고 보충 날짜/시간을 지정하면
- 해당 날짜의 비정규 목록에 자동 표시

#### 보충 당일 처리
- 완료: 학생이 등원하여 보충 수업 완료
- 미등원: 학생이 보충에도 불참 → 재지정 가능

---

## 7. 지연 항목 처리

### 지연이란?
예정일이 지났는데 아직 **미완료(pending)** 상태인 항목입니다.

### 확인 방법
- **오늘 날짜**를 선택했을 때만 지연 항목이 표시됩니다
- 목록 맨 위에 빨간색 "지연 — 미완료 (N건)" 구역에 나타남

### 처리 방법
```
방법 1: [재지정] 버튼 → 새 날짜 입력 → 저장
방법 2: 기타 → 확인 → 더 이상 관리하지 않음
```

### 자동 정리
- **1개월 이상** 지연된 항목은 시스템이 자동으로 "기타" 처리합니다

---

## 8. 확인 완료 후 재지정

확인 완료된 항목이라도 **미완료/미시행** 상태로 확인된 경우, 재지정이 가능합니다.

```
확인 완료 구역에서:
  [미완료] [재지정] [초기화]

  → [재지정] 클릭: 새 날짜/시간 입력
  → [초기화] 클릭: 미완료 상태로 되돌림
```

## 9. 등원 상태와 제한사항

### 등원 상태별 가능한 작업

| 등원 상태 | 완료/시행 | 미완료/미시행 | 기타 | 재지정 |
|----------|----------|-------------|------|-------|
| 미확인 (등원전) | ❌ 불가 | ✅ 가능 | ✅ 가능 | ✅ 가능 |
| 출석 | ✅ 가능 | ✅ 가능 | ✅ 가능 | ✅ 가능 |
| 지각 | ✅ 가능 | ✅ 가능 | ✅ 가능 | ✅ 가능 |
| 조퇴 | ✅ 가능 | ✅ 가능 | ✅ 가능 | ✅ 가능 |
| 결석 | ❌ 불가 | ✅ 가능 | ✅ 가능 | ✅ 가능 |

> **핵심 규칙**: 학생이 학원에 와야(출석/지각/조퇴) 완료/시행 처리를 할 수 있습니다.

## 10. 자주 묻는 질문

### Q. 학생이 안 왔는데 어떻게 하나요?
**A.** 미완료(미시행) → 확인 → 재지정으로 다음 등원일을 잡으세요.

### Q. 잘못 확인을 눌렀어요.
**A.** 확인 완료 구역에서 [초기화] 버튼을 누르면 원래대로 돌아갑니다.

### Q. 지연 항목이 안 보여요.
**A.** 오늘 날짜를 선택해야 지연 항목이 표시됩니다. 과거 날짜에서는 보이지 않습니다.

### Q. 진단평가 학생 정보를 수정하고 싶어요.
**A.** 목록에서 학생 이름 클릭 → 상세 패널 → 수정 버튼. 변경 이력이 자동으로 기록됩니다.

### Q. 더 이상 관리할 필요가 없는 항목은?
**A.** 기타 → 확인으로 처리하세요. 또는 1개월이 지나면 시스템이 자동 정리합니다.

### Q. 재지정하면 원래 기록은 사라지나요?
**A.** 아닙니다. 재지정 이력(이전 날짜, 새 날짜, 변경자, 사유)이 모두 저장됩니다.
